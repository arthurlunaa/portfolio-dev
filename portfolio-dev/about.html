Sou formado em Análise e Desenvolvimento de Sistemas pela Faculdade Unisa, com 5 anos de experiência em hardware e software de smartphones. Atualmente, estou expandindo meu conhecimento em desenvolvimento web, cursando Aplicação Web (HTML, CSS e JavaScript) no SENAI.
Minha especialidade e foco estão no front-end, com domínio em tecnologias como HTML5, CSS3 e JavaScript ES6+. Além disso, possuo habilidades em frameworks como React para construção de interfaces dinâmicas e Node.js para desenvolvimento de aplicações web baseadas em JavaScript. Também me interesso por design de interfaces e experiência do usuário (UX/UI), onde busco criar soluções responsivas e otimizadas tanto para desktops quanto para dispositivos móveis. Tenho uma forte paixão por novas tecnologias e tendências do mercado, como Progressive Web Apps (PWAs), Single Page Applications (SPAs), e estou sempre me atualizando sobre práticas como acessibilidade, performance e otimização de SEO. No futuro, meu objetivo é me especializar ainda mais em front-end, contribuir com inovação nas empresas de tecnologia e também explorar o desenvolvimento de apps móveis.
